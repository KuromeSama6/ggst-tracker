Guilty Gear -Strive- Tracker

A tracker for Guilty Gear -Strive-, with RATING UPDATE integration.
This project is open source. See it on Github.

First Time Setup

Enter your username as it appears in-game.
This will be used to acquire your data from RatingUpdate.

test

If your username doesn't work, try using your Rating ID.
Your Rating ID is found in your account's Rating Update page's URL.

For example, for http://ratingupdate.info/player/311F886E3ACC742/RA, the Rating ID is 311F886E3ACC742.
